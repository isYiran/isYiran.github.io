初学数学分析，很多概念理解的并不深刻，这份笔记只是初学阶段的一点整理。本篇为一些在数学分析中涉及的基本理论，包括集合论的基本概念与ZFC公理、数理逻辑基础，目的是为后续内容做好铺垫。

一、数理逻辑基础

这一块内容是为了整个笔记的完整性，以重述概念为主。

\S1.1 命题逻辑

Definition 1.1.1 命题是一个非真即假的陈述句。

通过逻辑连接符¬,∨,∧,→,↔\neg,\vee,\wedge,\to,\leftrightarrow¬,∨,∧,→,↔​​​，可以利用较简单的命题来构造复合命题，即

Definition 1.1.2 命题公式按如下规则生成：

  1. 命题变元是命题公式；
  2. 若PPP是命题公式，则¬P\neg P¬P是命题公式；
  3. 若P,QP,QP,Q​​是命题公式，则P∧Q,P∨Q,P→Q,P↔QP\wedge Q,P\vee Q,P\to Q,P\leftrightarrow QP∧Q,P∨Q,P→Q,P↔Q​​是命题公式；
  4. 当且仅当有限次应用1-3条规则得到的包含命题变元，逻辑联结词和圆括号的有意义的符号串是命题公式。

一个非常简单的递归定义，看上去还是非常清晰的。

Definition 1.1.3 若A→BA\to BA→B是一个永真式，则称其为永真蕴含式，记作A⇒BA\Rightarrow BA⇒B。

特殊地，若命题AAA​蕴涵着某个假命题，则AAA​为假，这即是反证法的思想：为了证明AAA为假，先假设AAA是真，再证明AAA蕴涵着某个一定为假的命题。

\S1.2 变量与量词

Definition 1.2.1 变量是表示某种特定类型的数学对象的符号，称未被指定为某个具体对象的变量为自由变量，否则称之为约束变量。

Definition 1.2.2 设P(x)P(x)P(x)是一个关于自由变量xxx​的命题。全称量词∀\forall∀读作“对任意的”，即“∀x∈X:P(x)\forall x\in X:P(x)∀x∈X:P(x)​为真”与“P(x)P(x)P(x)对所有的x∈Xx\in Xx∈X均为真”等价。类似地，存在量词∃\exists∃​读作“存在”。

多个量词可以嵌套在一起，否定一个命题会使得该命题中的全称量词变为存在量词，存在量词变为全程量词。

\S1.3 相等

相等遵守以下四条相等公理：

Axiom 1.3.1(自反公理) 给定任意的对象xxx​，有x=xx=xx=x​。

Axiom 1.3.2(对称公理) 给定任意两个同类型的对象x,yx,yx,y​ ,若x=yx=yx=y​，则y=xy=xy=x​。

Axiom 1.3.3(传递公理) 给定任意三个同类型的对象x,y,zx,y,zx,y,z​​，若x=yx=yx=y​​且y=zy=zy=z​，则x=zx=zx=z​。

Axiom 1.3.4(替换公理) 给定任意两个同类型的对象x,yx,yx,y​​，若x=yx=yx=y​​，则对任意一个函数或运算fff​​，有f(x)=f(y)f(x)=f(y)f(x)=f(y)​​，对任意一个关于xxx​​的性质P(x)P(x)P(x)​​，P(x)P(x)P(x)​​和P(y)P(y)P(y)​是等价的命题。

前三条公理非常显然，用下文提到的一个概念来描述即是，这三条公理断言了相等是一个等价关系。

而替换公理则说明了这些符号只是一个泛指的变量，例如在模3的意义下，111等于444，对于函数f(x)=x+2f(x)=x+2f(x)=x+2，就有1+21+21+2等于4+24+24+2。

数理逻辑暂且就只介绍这些粗浅的概念了，接下来引入一些集合论的概念。

二、集合论

下文不对公理和公理模式做区分。

\S2.1 朴素集合论

Definition 2.1.1(非正式) 一个集合是一些确定的对象无序的放在一起的整体，同时朴素集合论中的空集、子集和集合运算等概念就不再赘述了。

同时，为了更好的描述无限集合，朴素集合论承认概括公理：

Axiom 2.1.1(概括公理)(危险) 假设∀x\forall x∀x，存在关于xxx的性质P(x)P(x)P(x)，则存在一个集合{x∣P(x)\{x|P(x){x∣P(x)为真}\}}，使得∀y\forall y∀y，y∈{x∣p(x)y\in\{x|p(x)y∈{x∣p(x)为真}  ⟺  P(y)\}\iff P(y)}⟺P(y)为真。

概括公理实则是在断言每个性质都对应一个集合，由此我们可以简单地将满足某些性质的元素都归到一个集合中。

很可惜，朴素集合论遇上了麻烦。在这之前，我想先插入一些题外话。个人认为，“反例”在学习分析时，特别对初学者而言，是加深理解的一个利器。典型的例如Weierstrass函数，对分析学产生了重要的影响。而Russell即是构造了一个在概括公理下的“反例”，冲击了概括公理。

Russell悖论 令P(x)  ⟺  P(x)\iffP(x)⟺​​"xxx​​是一个集合，并且x∉xx\notin xx∈/​x​​"，定义A≔{x∣P(x)A\coloneqq \{x|P(x)A:={x∣P(x)​为真}\}}​​。假如A∉AA\notin AA∈/​A​​​，则P(A)P(A)P(A)​为真，从而A∈AA\in AA∈A​；假如A∈AA\in AA∈A​，则P(A)P(A)P(A)​为假，从而A∉AA\notin AA∈/​A。因此，无论哪种情况，都有A∈AA\in AA∈A和A∉AA\notin AA∈/​A​​​。

这个悖论还有一个通俗版本叫理发师悖论。产生这个悖论的原因在于概括公理允许过大的集合存在，因此需要引入一系列公理限制集合的构造（题外话：个人认为第三次数学危机并没有完全消除）。下面介绍ZFC公理集合论。（再插句题外话：为什么朴素集合论没有那么多的公理呢？因为在假定概括公理为真的情况下，可以轻松推出下文许多公理，很可惜，这么美好的事情是不现实的。）

\S2.2 ZFC集合论

先从定义集合的相等开始：

Axiom 2.2.1(外延公理) ∀x(x∈A↔x∈B)↔A=B\forall x(x\in A\leftrightarrow x\in B)\leftrightarrow A=B∀x(x∈A↔x∈B)↔A=B​

即称两个集合A,BA,BA,B是相等的，当且仅当AAA与BBB中的元素相同。

※ 这个公理说明集合内的元素是集合唯一的性质。

Axiom 2.2.2(分类公理) ∀A∃B∀x(x∈B↔x∈A∧P(x))\forall A\exists B\forall x(x\in B\leftrightarrow x\in A\wedge P(x))∀A∃B∀x(x∈B↔x∈A∧P(x))​​​

即设A是一个集合，P(x)P(x)P(x)​​是AAA​​中元素的一个性质，则存在集合{x∈A∣P(x)}\{x\in A|P(x)\}{x∈A∣P(x)}​为所有AAA中具有性质PPP的元素。

分类公理即对概括公理的弱化，假如集合真的存在（注意，目前我们还不能断定真的存在集合），我们已经能构造出空集了。

Proposition 2.2.1 若存在一个集合，则唯一存在没有任何元素的集合。

  • ***Proof***​​ 假设存在一个集合AAA​，令P(x)P(x)P(x)​为x≠xx\neq xx​=x​​，则由分类公理即可得存在性，再有外延公理可得唯一性。□\Box□

外延公理已经定义了集合的相等，回想下\S1.3​中关于相等的替换公理，我们来验证下替换公理适用于分类。

  • Proof 假设替换公理不适应，即存在A=A′A=A'A=A′，有{x∈A∣P(x)}≠{x∈A′∣P(x)}\{x\in A|P(x)\}\neq\{x\in A'|P(x)\}{x∈A∣P(x)}​={x∈A′∣P(x)}​。记两个集合分别为B,B′B,B'B,B′，不妨设BBB中存在元素xxx，有x∉B′x\notin B'x∈/​B′。

    由x∈Bx\in Bx∈B​​​​可知x∈Ax\in Ax∈A​​​​。又由外延公理，x∈A′x\in A'x∈A′​​​​。假设P(x)P(x)P(x)​​​​为真，则x∈B′x\in B'x∈B′​​​​，否则有x∉B′x \notin B'x∈/​B′​​​​，均与上述矛盾，从而替换公理适用于分类。□\Box□​​​

现在集合还很匮乏，为了构造更多可用的集合，配对“公理”被引入了。

Axiom 2.2.3(配对"公理") ∀A∀B∃C∀x(x∈C↔x=a∨x=b)\forall A\forall B\exists C\forall x(x\in C\leftrightarrow x=a\vee x=b)∀A∀B∃C∀x(x∈C↔x=a∨x=b)

即对对象A,BA,BA,B​，存在集合C={A,B}C=\{A,B\}C={A,B}​​。

Axiom 2.2.4(并集公理) ∀A∃B∀x(x∈B↔∃y(x∈y∧y∈A))\forall A\exists B\forall x(x\in B\leftrightarrow\exists y(x\in y\wedge y\in A))∀A∃B∀x(x∈B↔∃y(x∈y∧y∈A))​

即对AAA，存在一个集合BBB，AAA中元素的元素都在BBB中且BBB中元素都在AAA的元素中，并引入记号B=⋃AB=\bigcup AB=⋃A。

集合的交也可被类似定义。

虽然现在还没有定义函数，但可以暂用一下以前的知识。为了考察定义域为XXX，值域为YYY的所有函数，我们需要考虑XXX的所有子集和YYY的所有子集，因此引入了幂集公理。

Axiom 2.2.5(幂集公理) ∀A∃B∀x(x∈B↔x⊆A)\forall A\exists B\forall x(x\in B\leftrightarrow x\subseteq A)∀A∃B∀x(x∈B↔x⊆A)

即对集合AAA，存在集合BBB，其元素是AAA的所有子集。

注意，上述所有的定理都是在假设有集合存在的情况下，目前我们还不能保证确实存在有集合。先引入两个定义：

Definition 2.2.1 对任意集合AAA，A+≔A∪{A}A^+\coloneqq A\cup\{A\}A+:=A∪{A}称为AAA的后继集。

Definition 2.2.2 Inf(x)↔∅∈x∧∀u(u∈x→u+∈x)Inf(x)\leftrightarrow\emptyset\in x\wedge\forall u(u\in x\to u^+\in x)Inf(x)↔∅∈x∧∀u(u∈x→u+∈x)，将性质InfInfInf为真的集合称为归纳集。

对于一个集合，可以进行的一个最基本的操作是取出集合中的每一个元素，按某种方式将每个元素都转换成一个新 对象，并最好能将新的对象都放在一个集合里。为此，引入替代公理。

接下来引入无穷公理来说明集合存在。

Axiom 2.2.6(无穷公理) ∃A(∃∅(∅∈A)∧∀x(x∈A→x+∈A))\exists A(\exists \emptyset(\emptyset\in A)\wedge\forall x(x\in A\to x^+\in A))∃A(∃∅(∅∈A)∧∀x(x∈A→x+∈A))​

即空集是存在的，且存在归纳集AAA，包括某个存在的集合和任何其中的集合的后继集。

※ 无穷公理先说明空集存在，再说明无穷集存在。

这时已经确保了有集合存在，因此空集也被唯一确定了，记其为∅\emptyset∅，把这个结论称为空集定理。
